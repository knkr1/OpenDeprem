# Open Deprem (IN DEVELOPMENT)

An open source web based earthquake notifier and alerter.

## Tools

Based on React, Leaflet, Tailwind CSS. 

## Setup

Install the dependencies
```npm install```

Developing
```npm run start```

Building
```npm run build```
